ICON FILES NEEDED
==================

Please create these icon files for full PWA support:

1. apple-touch-icon.png (180x180px)
   - Use the baby face design from favicon.svg
   - Pink to purple gradient background
   - Should be visible on iOS home screen

2. icon-192.png (192x192px)
   - Android home screen icon
   - Same baby face design

3. icon-512.png (512x512px)
   - High-res version for Android
   - Same design, higher quality

4. og-image.jpg (1200x630px)
   - For social media sharing (Facebook, Twitter)
   - Include app title and tagline
   - Show game screenshot or baby illustration

5. screenshot.jpg (optional, for app stores)
   - Shows the game in action

You can use tools like:
- Canva (canva.com)
- Figma (figma.com)  
- Or any image editor

For now, the favicon.svg will be used as fallback.
